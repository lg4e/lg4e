2. 为什么采用卡片布局

LG4E的网站布局源自 Tsugi project ，类似的网站还有 PY4E 、CC4E 等。
在学习 Python 的过程中，PY4E 的卡片式设计给了我一些启发——我就想，是不是任何领域的知识都可以通过这种方式，简单而清晰地传达给所有人。LG4E正是基于这种想法所做的一次尝试。

3. 如果你也有兴趣

LG4E在外观上沿用了 Tsugi project 的风格，但采用了不同的技术路线。
你可以在 GitHub 上获取到 LG4E 的所有代码。这意味着你可以自由复制，发布并按照你的需求进行修改与重组。
